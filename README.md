# Консольный TCP сервер
Простой TCP-сервер на Windows, который принимает подключения клиентов, получает данные и отправляет их обратно. В текущей версии реализована многопоточность для обслуживания нескольких клиентов одновременно и базовый broadcast сообщений между клиентами. В дальнейшем планируется хранение истории сообщений, никнеймы,  логирование, команды и системные сообщения, разделение логики сервера и клиента.
