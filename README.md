# Консольный TCP сервер
Простой TCP-сервер на Windows, который принимает подключения клиентов, получает данные и отправляет их обратно. В текущей версии реализована многопоточность для обслуживания нескольких клиентов одновременно и счётчик подключений. В дальнейшем планируется хранение истории сообщений и базовая структура чата.
